[笔记]-SpringMVC-文件上传

2017-11-25 10:24

使用SpringMVC

使用SpringMVC的CommonsMultipartFile类读取文件
需要用到上传文件的两个jar包 commons-logging.jar、commons-io-xxx.jar

未选择上传文件就提交表单，处理器方法的 MultipateFile 形参所接收到的实参值并非 null，而是 empty 文件。
判断条件为 file.isEmpty()，而非 file == null。
对于上传多个文件，即使没有选择任何要上传的文件，MultipartFile 数组也不为 null。
其 length 值也大于 0。
只能逐个文件表单元素判断，判断文件是否为 empty。

上传单个文件时 MultipartFile 参数不用使用@RequestParam 修饰，上传多个文件时 MultipartFile[]需要@RequestParam 修饰。
上传多个文件时，每个表单中的文件对象框架均会将其转换为MultipartFile类型，处理器方法需要的不是 MultipartFile 类型参数，而是 MultipartFile[]数组类型。
默认情况下框架只会将一个一个的表单元素转换为一个一个的对象，但不会将这多个对象创建为一个数组对象。
@RequestParam 修饰，表单传来的参数与处理器方法接收的参数名称与类型相同，框架调用相应的转换器转换。

[笔记]-Servlet3.0

2017-11-27 9:24

web.xml

Servlet3.0提供了注解(annotation)，不再需要在web.xml文件中进行Servlet的部署描述
@WebServlet描述一个Servlet

@WebServlet(name = "Servlet3", urlPatterns = {"/Servlet3", "/Servlet35"})
@WebServlet(name = "Servlet3", value = {"/Servlet3", "/Servlet35"})

Servlet3.0提供@WebFilter注解将一个实现了javax.servlet.Filter接口的类定义为过滤器，不再需要在web.xml文件中配置过滤器。

Servlet3.0提供@WebListener注解将一个实现了监听器接口的类定义为监听器，不再需要在web.xml文件中配置监听器。

Servlet2.5中，实现文件上传一般都需要第三方开源组件，如Apache的commons-fileupload组件，在Servlet3.0中提供了对文件上传的原生支持。

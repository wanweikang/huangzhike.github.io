控制整个JSP是否执行

多个标签库.tld，标签库的uri不能设置成相同的。

引用mmp.tld标签库：<%@taglib uri="/WEB-INF/mmp.tld" prefix="mmp" %>

引用simpletag.tld标签库：<%@taglib uri="/WEB-INF/simpletag.tld" prefix="mmp" %>

  • 在标签处理器中如果没有调用JspFragment.invoke方法，相当于忽略标签体内容；

  • 在标签处理器中重复调用JspFragment.invoke方法，则标签体内容将会被重复执行；

  • 若想在标签处理器中修改标签体内容，只需在调用invoke方法时指定一个可取出结果数据的输出流对象（如StringWriter），让标签体的执行结果输出到该输出流对象中，然后从该输出流对象中取出数据进行修改后再输出到目标设备。

通过标签的属性控制标签体的执行次数

如果标签的属性值是8种基本数据类型，JSP页面在传递字符串时，会自动转换成相应的类型，如果标签的属性值是复合数据类型，无法自动转换。

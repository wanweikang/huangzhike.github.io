[笔记]-集合-List

2017-10-12 16:35

  • Collection 接口是最基本的集合接口，List 和 Set 都是继承自 Collection 的“子接口” 。

    • List 接口：有序（根据索引访问），可重复
    • Set 接口：无序，不可重复，重复靠hasCode与equals方法区分

  • 实现 List 接口的子集合主要有：

    • ArrayList（异步，线程不安全，动态数组）
    • LinkedList
    • Vector（同步，线程安全的动态数组）
    • Stack

ArrayList 是动态数组，每个 ArrayList 都有初始容量（10），每次向容器中增加元素的同时都会进行容量检查，快溢出时，就会扩容。
最好指定一个初始容量值，避免过多扩容。

ArrayList是动态数组，LinkedList 是双向链表。
它除了有 ArrayList 的基本操作方法外还额外提供了 get，remove，insert 方法。
LinkedList 不能随机访问，所有操作按双重链表执行。
在列表中索引的操作将从开头或结尾遍历列表（从靠近指定索引的一端）。
好处就是可以通过较低的代价在 List 中进行插入和删除操作。
与 ArrayList 一样，LinkedList 也是非同步的，如果多个线程同时访问一个 List，必须自己实现访问同步。

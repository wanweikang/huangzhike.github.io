[笔记]-集合-Set

2017-10-12 16:35
  • 实现了 Set 接口的集合有：EnumSet、HashSet、TreeSet。
    • EnumSet 所有的元素都是枚举类型。
    • HashSet 查询速度最快的集合，内部以 HashCode 实现。内部元素的顺序由哈希码来决定，不保证 set 的迭代顺序，不保证该顺序不变。
    • TreeSet 使用元素的自然顺序对元素进行排序，或根据创建 Set 时的 Comparator 进行排序，具体取决于使用的构造方法。

[笔记]-集合-Set

2017-10-12 16:35
  • Set 接口：无序，不可添加重复元素，重复靠hasCode与equals方法区分。
    • 实现了 Set 接口的集合有：EnumSet、HashSet、TreeSet。
      • EnumSet：所有的元素都是枚举类型。
      • HashSet：查询速度最快的集合，内部元素的顺序由HashCode决定，乱序迭代。
      • TreeSet：有序。

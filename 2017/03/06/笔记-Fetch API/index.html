[笔记]-Fetch API

2017-03-06 20:12:31

来源：http://blog.percymong.com/2017/03/04/fetch-api/

XMLHttpRequest 常用它来执行异步请求，但是它在设计上不符合职责分离原则，将输入、输出和用事件来跟踪的状态混杂在一个对象里， Fetch API 旨在修复这些缺陷。

  • Fetch API 接口
    • GlobalFetch.fetch()： 用于发起获取资源的请求，返回一个 promise 对象；
    • Headers 对象： 对HTTP request 和 response headers 执行各种操作；
    • Request 对象： 用来表示资源请求；
    • Response 对象： 呈现了对一次请求的响应数据。

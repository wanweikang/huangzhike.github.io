RESTful架构对URL进行规范

  • 一般的URL：http://.../xxx.action?id=001&type=aaa
  • REST的URL：http://.../xxx/001
<servlet>
    <servlet-name>springmvc_rest</servlet-name>
    <servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>
    <init-param>
        <param-name>contextConfigLocation</param-name>
        <param-value>classpath:spring/springmvc.xml</param-value>
    </init-param>
</servlet>

<servlet-mapping>
    <servlet-name>springmvc_rest</servlet-name>
    <!-- 对静态文件需要配置不让DispatcherServlet解析，实现RESTful -->
    <url-pattern>/</url-pattern>
</servlet-mapping>

添加JSON转换的依赖Jackson，配置JSON转换器<mvc:annotation-driven/>…



function requestJSON() {
    $.ajax({
        type:'post',
        url:'${pageContext.request.contextPath}/requestJSON.action',
        contentType:'application/json;charset=utf-8',
        data:'{"name":"手机", "price":999}',
        success:function(data) { // 返回JSON结果
            console.log(data);
        }
    });
}
@RequestMapping("/requestJSON")
public @ResponseBody ItemsCustom requestJSON(@RequestBody ItemsCustom itemsCustom) {
    return itemsCustom;
}
// @RequestBody：将请求的JSON转成itemsCustom对象
// @ResponseBody：将itemsCustom转成JSON
function responseJSON() {
    $.ajax({
        type:'post',
        url:'${pageContext.request.contextPath}/responseJSON.action',
        // 请求是key/value不需要指定contentType，默认就是key/value
        // 默认Content-Type是application/x-www-form-urlencoded
        data:'name=手机&price=999',
        success:function(data) { // 返回JSON
            console.log(data.name);
        }
    });
}
@RequestMapping("/responseJSON")
public @ResponseBody ItemsCustom responseJSON(ItemsCustom itemsCustom) {
    // @ResponseBody将Java对象itemsCustom转成JSON输出
    return itemsCustom;
}

另一个栗子：

// {id}表示占位符，对应@PathVariable标记的形参，如果形参和占位符的变量一样，不用指定
@RequestMapping("/itemsView/{id}")
public @ResponseBody ItemsCustom itemsView(@PathVariable("id") Integer id) {
    ItemsCustom itemsCustom = itemsService.findItemsById(id);
    return itemsCustom;
}

http://localhost:8080/SpringMVC/itemsView/1返回一串JSON数据。

@RestController
public class BookController {

    @Resource
    private BookService bookService;

    @GetMapping("/books")
    public List<Book> books(){
        return bookService.findAll();
    }
}

@RestController相当于@ResponseBody＋@Controller。

@RestController注解Controller中的方法无法返回.jsp页面，视图解析器InternalResourceViewResolver不起作用，返回的内容就是Return的内容。

例如：本来到success.jsp，则显示success。

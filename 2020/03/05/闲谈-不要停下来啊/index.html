[闲谈]-不要停下来啊

2020-03-05

没想到20年的第一篇文章竟然是这个。。。

我忏悔，我有罪。

本来打算去年年底辞职后给自己放两个月假，并且准备面试，按理说时间绰绰有余。

结果：

先玩就是了，反正时间很充裕 -> 继续玩 -> 明天再学习一下吧 -> 反正还有时间，不急 -> 不管，继续玩 -> 看完这部XXX我就去复习 -> 看完这部YYY我就去复习 -> 看完这部ZZZ我就去复习 -> 还想继续看？？？，反正复习都是之前学过的东西，不会很耗时间的，明天再复习吧 -> 明天再复习 -> 明天我一定去复习 -> 明天一定… -> WDNMD，已经2月了！

所以这段时间我干的事情就是彻底放飞自我，熬夜追番追漫追小说，疯狂的时候甚至连续几天通宵到凌晨五六点，天，现在想想自己都害怕，一松懈下来，就光速堕落成废人，但是做尼特实在太舒服了我有什么办法！

我本质上就是一个极其懈怠的人，平时上班时，会注意保持比较好的节奏和规律，加上的确是挺感兴趣的，有目标也有规划，所以可以一直坚持学习也不觉得累，但是一停下来，哦吼，凉凉。

だからよ、止まるんじゃねぇぞ👆・・・

现在对自己的本质有了深刻的认识。

原因：

  1. 我本质懈怠，是编码在DNA里的懒惰，懒癌晚期 + 拖延症晚期；

  2. 惯性这个东西很重要，一旦停下来时间久了，真的很难搞；

  3. 环境因素，宅在家中实在很没动力学习，氛围很重要，选择重点高中大学什么的，也是类似的原因，如果周围的人整天玩玩玩，你也很难专注学习，在自习室的学习效率和干劲肯定比在宿舍高；

  4. 我喜新厌旧，真的是讨厌复习，不喜欢看旧的东西（虽然并不是很熟，也喜欢新的，而不是二刷）；

  5. 没有一个强制的Deadline和规划执行，不说了，现在还是疫情封锁中呢（还敢找借口？）。

现在强制每天下午和晚上固定时间用来复习，什么时候必须完成哪个部分的内容。时间有点赶，本来打算盖一个精装别墅的，结果成了烂尾楼。

我是真的讨厌复习。。。特别是死记硬背。。。算了，一直不都是裸考过来的吗，复习有毛用啊。

不喜欢也要看一下啊。

妈妈我再也不敢了。

然后说说这段时间的经历吧，年后2月中开始准备简历，一直是以某大厂为目标的，于是找了内推，一开始不知死活地投了某部门，然后内推的大佬告诉我，项目没啥深度，成功几率不大。

好吧，于是又尝试了另外一个部门，没想到居然愿意给我一个机会，于是乎，居然面到了第四轮，就在我差点以为有希望的时候，额，挂了。

哈哈，我问了内推人，大概是项目经验浅薄的原因，嗯，的确是，毕竟我工作经验才两年，而且待的都是小作坊，不过这次也知道自己的不足之处了，很宝贵的经验。

然后开始在某勾和某Boss投简历，不知道是我的简历太差，还是因为疫情，大部分没啥回音。。。

然后就是开始有点后悔裸辞了，还嗨了这么久。。。

然后是新公司，业务是电商团购，我是挺感兴趣的，不过估计以后都会挺忙的，经常加班，不会像去年那样，经常有空学别的东西。

那段时间其实还是打下了很好的基础的，以后博客可能更新得比较慢，但是还是会继续更新的，毕竟还有很多想学要学的东西，比如数据库，中间件，注册中心这些，还是以原理逻辑和源码为主。

嗯嗯，大概就是这样。

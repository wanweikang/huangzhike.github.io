2016-10-27

Number对象的属性。

Number.POSITIVE_INFINITY // Infinity
Number.NEGATIVE_INFINITY // -Infinity
Number.NaN // NaN
Number.MAX_VALUE // 1.7976931348623157e+308
Number.MIN_VALUE // 5e-324

Number对象实例的方法：Number.prototype.toString()。

(10).toString() // "10"
(10).toString(2) // "1010"
(10).toString(8) // "12"
(10).toString(16) // "a"

不加括号，点会被解释成小数点。

10.toString(2) // SyntaxError: Unexpected token ILLEGAL

加两个点，第一个点理解成小数点（即10.0），第二个点理解成调用对象属性。

10..toString(2)  // "1010"

可以直接对一个小数使用toString方法。

10.5.toString() // "10.5"
10.5.toString(2) // "1010.1"

通过方括号运算符也可以调用toString方法。

10['toString'](2) // "1010"

Number.prototype.toFixed()：将一个数四舍五入为指定小数位数。

(10).toFixed(2) // "10.00"
(10.005).toFixed(2) // "10.01"

Number.prototype.toExponential()：将一个数转为科学计数法形式。

(10).toExponential(1) // "1.0e+1"
(1234).toExponential(1) // "1.2e+3"

Number.prototype.toPrecision()：将一个数转为指定位数的有效数字。

(12.34).toPrecision(1) // "1e+1"
(12.34).toPrecision(2) // "12"
(12.34).toPrecision(3) // "12.3"
(12.34).toPrecision(4) // "12.34"
(12.34).toPrecision(5) // "12.340"

toPrecision方法用于四舍五入时不可靠，可能跟浮点数不是精确储存有关。

(12.35).toPrecision(3) // "12.3"
(12.15).toPrecision(3) // "12.2"

实例方法：Number.prototype对象上自定义方法，被Number的实例继承。

Number.prototype.add = function (x) {
    return this + x;
};

Number对象的实例就是数值，在数值上调用某个方法，数值会自动转为对象。

8['add'](2) // 10
(8).add(2) // 10

add方法返回的还是数值，所以可以链式运算。

Number.prototype.subtract = function (x) {
    return this - x;
};

(8).add(2).subtract(4) // 6

将一个数值自动遍历为一个数组。

Number.prototype.iterate = function () {
    var result = [];
    for (var i = 0; i <= this; i++) {
        result.push(i);
    }
    return result;
};

(8).iterate() // [0, 1, 2, 3, 4, 5, 6, 7, 8]

实例方法，只能定义在它的原型对象Number.prototype上，数值本身是无法自定义属性的。

var n = 1;
n.x = 1;
n.x // undefined

Math对象是JavaScript的内置对象，提供一系列数学常数和数学方法。

Math对象不是构造函数，不能生成实例，所有的属性和方法都必须在Math对象上调用。

  • E：常数e。
  • LN2：2的自然对数。
  • LN10：10的自然对数。
  • LOG2E：以2为底的e的对数。
  • LOG10E：以10为底的e的对数。
  • PI：常数Pi。
  • SQRT1_2：0.5的平方根。
  • SQRT2：2的平方根。
Math.E // 2.718281828459045
Math.LN2 // 0.6931471805599453
Math.LN10 // 2.302585092994046
Math.PI // 3.141592653589793

舍入方法

  • Math.ceil()：向上取整。
  • Math.floor()：向下取整。
  • Math.round()：四舍五入取整。
Math.round(0.1) // 0
Math.round(0.5) // 1
// 负值与正值不同，如 .5的处理
Math.round(-1.1) // -1
Math.round(-1.5) // -1

Math.abs()：返回绝对值。

Math.max(2, -1, 3) // 3
Math.min(2, -1, 1) // -1

// 要找到数组中的最大或最小值，使用apply()方法：
var array = [1, 2, 3, 4];
Math.max.apply(null, array);

Math.pow()：返回以第一个参数为底数、第二个参数为幂的指数值。

Math.pow(2, 2) // 4
Math.pow(2, 3) // 8

Math.sqrt()：返回参数值的平方根。如果参数是一个负值，返回NaN。

Math.sqrt(4) // 2
Math.sqrt(-4) // NaN

Math.log()：返回以e为底的自然对数值。

Math.log(Math.E) // 1
Math.log(10) // 2.302585092994046
// 求以10为底的对数，可以除以 Math.LN10；
Math.log(100)/Math.LN10 // 2
// 求以2为底的对数，可以除以 Math.LN2
Math.log(8)/Math.LN2 // 3

Math.exp()：返回常数e的参数次方。

Math.exp(1) // 2.718281828459045
Math.exp(3) // 20.085536923187668

Math.random()：返回0到1之间的伪随机数，可能等于0，一定小于1。

// 返回给定范围内的随机数
function getRandomArbitrary(min, max) {
    return Math.random() * (max - min) + min;
}
// 返回给定范围内的随机整数
function getRandomInt(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;
}
parseInt(Math.random() * 8) // [0,8)的随机数 ， 能取到0，但是不能取到8。就是[0,7]

parseInt(Math.random() * 8) + 3 // [3,11)的随机数，能取到3，但是不能取到11。就是[3,10]

parseInt(Math.random() * 9) + 3 // [3,11]的随机数，能取到3，能取到11

parseInt(Math.random() * (max - min + 1)) + min; // 在[min,max]闭区间取随机数

Math.floor(Math.random() * (max - min + 1)) + min;
// 从数组中随机获取成员
var arr = [1, 2 , 3 , 4, 5 ];
arr[Math.floor(Math.random() * arr.length)];

var arr = [] , max = 100;
for(var i=1; arr.push(i++) < max;);

三角函数方法：sin方法返回参数的正弦，cos方法返回参数的余弦，tan方法返回参数的正切。

Math.sin(0) // 0
Math.cos(0) // 1
Math.tan(0) // 0

asin方法返回参数的反正弦，acos方法返回参数的反余弦，atan方法返回参数的反正切。三个方法的返回值都是弧度值。

Math.asin(1) // 1.5707963267948966
Math.acos(1) // 0
Math.atan(1) // 0.7853981633974483

数值相关的全局方法

parseInt方法将字符串或小数转化为整数，截断小数点后面的任何东西，不四舍五入。

parseInt("1.2") // 1
parseInt(1.2) // 1

如果字符串头部有空格，空格会被自动去除。

parseInt('   8') // 8

如果字符串包含不能转化为数字的字符，则不再转化，返回完成部分。

parseInt("8a") // 8
parseInt("12**") // 12

如果字符串的第一个字符不能转化为数字（正负号除外），转换失败，返回NaN。

parseInt("abc") // NaN
parseInt(".3") // NaN
parseInt("") // NaN

parseInt方法接受第二个参数（2到36），表示第一个参数的进制。默认十进制。

用parseInt方法进行进制转换（转换为十进制）。

parseInt(1000, 2) // 8
parseInt(1000, 6) // 216
parseInt(1000, 8) // 512
parseInt("1011", 2) // 11

如果第一个参数是字符串则进行装换，数字则不装换。

parseInt('FF',10); // NaN
parseInt('FF',16); // 255
parseInt(010, 10) // 8
parseInt("010",10) // 10
parseInt(010, 8) // NaN
parseInt("010",8) // 8
parseInt(020, 10) // 16
parseInt(020, 8) // 14

parseInt的很多复杂行为，都是由八进制的前缀0引发的。

因此，ES5不允许parseInt将带有前缀0的数字，视为八进制数。

对于会自动转为科学计数法的数字，parseInt会出现奇怪的错误。

parseInt(1000000000000000000000.5, 10) // 1
// 等于
parseInt('1e+21', 10) // 1

parseInt(0.0000008, 10) // 8
// 等于
parseInt('8e-7', 10) // 8

parseFloat方法将字符串转为浮点数，如果字符串包含不能转化为浮点数的字符，不再转化，返回已转好部分。

parseFloat("3.14"); // 3.14
parseFloat("314e-2"); // 3.14
parseFloat("0.0314E+2"); // 3.14
parseFloat("3.14abc"); // 3.14
parseFloat("3.14.123")   //3.14

parseFloat方法自动过滤字符串前导的空格。

parseFloat("\t\v\r12.34\n ") // 12.34

如果第一个字符不能转化为浮点数，则返回NaN。

parseFloat("FF2") // NaN
parseFloat("") // NaN

parseFloat将空字符串转为NaN，不同于Number函数。

parseFloat(true)  // NaN
Number(true) // 1

parseFloat(null) // NaN
Number(null) // 0

parseFloat('') // NaN
Number('') // 0

parseFloat('123.45#') // 123.45
Number('123.45#') // NaN

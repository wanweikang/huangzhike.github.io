[笔记]-字符串

2016-10-27

2016-11-18重新整理

如果'本身也是字符，用""括起来，字符串内部既包含'又包含"用转义字符\标识，如：

'I\'m \"OK\"!'; // I'm "OK"!
  • 转义字符\，如\n表示换行，\t表示制表符，字符\本身也要转义，\\表示的字符是\。
  • ASCII字符可以以\x##形式的十六进制表示，如：\x41等同于A。
  • \u####表示一个Unicode字符：\u4e2d\u6587等同于中文。
  • 多行字符串用\n比较费事，ES6新增了多行字符串表示方法，`…`表示：
`这是一个
多行
字符串`;

连接多个字符串用+。ES6新增模板字符串，表示方法和多行字符串一样，但会自动替换字符串中的变量：

var name = '小明';
var age = 20;
var message = `你好, ${name}, 你今年${age}岁了!`;
message; // "你好, 小明, 你今年20岁了!"

下面字符串方法，不会改变原字符串，而是返回新字符串：

var s = "Hello";
s.toUpperCase(); // 返回"HELLO"
s.indexOf("world"); // 返回7
s.indexOf("World"); // 大小写敏感，没有找到指定的子串，返回-1
s.lastIndexOf("World",7) // 6，由索引7开始从尾到头搜索
s.substring(0, 3); // 从索引0开始到3（不包括3），返回"hel"
s.substring(7); // 从索引7开始到结束，返回"world"
s.substr(0, 3); // 从索引0开始取3个字符（不建议使用）
s.replace("l","0"); // He0lo 只替换1个
var a = "1h2h3h4h5";
a.split("h"); // ["1", "2", "3", "4", "5"]，字符串 数组
a.slice(-3,-1); // 4h，倒数第三个到倒数第一个（不包括）

slice,substr和substring

  • 都接收两个参数
    • slice和substring接收的是起始位置和结束位置(不包括结束位置)；
    • substr接收的则是起始位置和所要返回的字符串长度（类似splice）；
    • substring是以两个参数中较小一个作为起始位置，较大的参数作为结束位置；
  • 当接收的参数是负数
    • slice会将它字符串的长度与对应的负数相加，结果作为参数；
    • substr则仅仅是将第一个参数与字符串长度相加后的结果作为第一个参数；
    • substring则将负参数直接转换为0；
  • 超出范围索引不报错，返回undefined
var s = "Hello, world!";
s.length; // 13
s[0]; // "H"
s[6]; // " "
s[13]; // undefined

  • String.prototype.charCodeAt() 获取某位置上的字符编码
  • String.prototype.slice() 截取字符串
  • String.prototype.subString() 截取字符串

  • String.prototype.indexOf() 获取字符在字符串中的位置
  • String.prototype.lastIndexOf() 获取字符在字符串中的位置 逆向
  • String.prototype.search() 获取字符在字符串中的位置
  • String.prototype.split() 字符串转化成数组
  • String.prototype.replace() 字符串替换
  • String.prototype.trim() 返回一个字符串的副本，删除前置及后缀的所有空格

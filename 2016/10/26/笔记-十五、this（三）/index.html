[笔记]-十五、this（三）

2016-10-26 18:45

Node.js
Node.js中，this的指向分成两种情况。
全局环境中，this 指向全局对象 global；模块环境中，this指向 module.exports。

ES6中引入了箭头函数。
箭头函数不用 function 关键字定义，而使用“胖箭头”的操作符 => 定义。
箭头函数不使用 this 的规则，而是根据外层（函数或者全局）作用域来决定 this 。

foo() 内部箭头函数会捕获调用时 foo() 的 this 。
由于 foo() 的 this 绑定到 obj1 ， bar （引用箭头函数）的 this 也会绑定到 obj1 ，绑定无法被修改。（new 也不行）

箭头函数可以像 bind() 一样确保函数的 this 被绑定到指定对象，它用更常见的词法作用域取代了传统的 this 机制。

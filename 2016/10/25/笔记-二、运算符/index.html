中括号运算符

如果对象的属性本身就包含点 (a.b)，则使用中括号运算符来访问：obj[“a.b”]

圆括号运算符

把表达式放在圆括号之中，作用是求值，返回表达式的值。

把对象放在圆括号之中，会返回对象的值，即对象本身。

将函数放在圆括号中，返回函数本身。

如果圆括号紧跟在函数的后面，作用是调用函数。即对函数求值。

圆括号的作用是求值，如果将语句放在圆括号之中，就会报错，因为语句没有返回值。

void运算符

执行一个表达式，然后返回undefined。

主要用于书签工具（bookmarklet）或者超链接中插入代码，目的是返回undefined防止网页跳转。

常用于网页链接的鼠标点击事件写法。

逗号运算符
对两个表达式求值，并返回后一个表达式的值。

左结合与右结合
对于优先级别相同的运算符，大多数情况，计算顺序总是从左到右，这叫做运算符的“左结合”（left-to-right associativity）
x + y + z

少数运算符计算顺序是从右到左，这叫做运算符的“右结合”（right-to-left associativity）。主要是赋值运算符（=）和三元条件运算符（?:）。

逻辑运算符

且运算符（&&）

  • 如果第一个为true，则返回第二个的值（是值，不是布尔值）；
    "a" && (1+2) // 3
  • 如果第一个为false，直接返回第一个的值（是值，不是布尔值），不再对第二个运算子求值。

这种跳过第二个运算子的机制，被称为“短路”。

或运算符（||）

  • 如果第一个为true，直接返回第一个的值，且不再对第二个运算子求值；
    "a" || "b" // "a"
  • 如果第一个为false，则返回第二个运算子的值。
    "" || "b" // "b"

或运算符多个连用，返回第一个布尔值为true的表达式的值。

或运算符可以为一个变量设置默认值。

总结：
a&&b， a真，那么扔b；如果a假，扔a
a||b， a真，那么扔a；如果a假，扔b

位运算符

位运算符用于直接对二进制位进行计算，共有7个。

  • 或运算（or）：符号为|，表示两个二进制位中有一个为1，则结果为1，否则为0。
  • 与运算（and）：符号为&，表示两个二进制位都为1，则结果为1，否则为0。
  • 否运算（not）：符号为～，表示将一个二进制位变成相反值。
  • 异或运算（xor）：符号为?，表示两个二进制位中有且仅有一个为1时，结果为1，否则为0。
  • 左移运算（left shift）：符号为<<，
  • 右移运算（right shift）：符号为>>，
  • 带符号位的右移运算（zero filled right shift）：符号为>>>，

在JavaScript内部，数值都是以64位浮点数的形式储存，但做位运算时，以32位带符号的整数进行，并且返回值也是一个32位带符号的整数。

位运算符只对整数起作用，如果不是整数，会将小数部分舍去，只保留整数部分
所以，将一个小数与0进行或运算，等同于对该数去除小数部分，即取整数位。

这种取整方法不适用超过32位整数最大值2147483647的数。

或运算
0 | 3 // 3
与运算
0 & 3 // 0

否运算
将每个二进制位都变为相反值（0变为 1，1变为 0）。
~ 3 // -4

3是 00000000000000000000000000000011，
否运算以后得到 11111111111111111111111111111100，
由于第一位是 1，所以这个数是一个负数。
JavaScript内部采用2的补码形式表示负数，即需要将这个数减去1，再取一次反，然后加上负号，才能得到这个负数对应的10进制值。
这个数减去1等于 11111111111111111111111111111011，再取一次反得到 00000000000000000000000000000100，再加上负号就是-4。
简单记忆成，一个数与自身的取反值相加，等于-1。
~ -3 // 2
上面表达式可以这样算，-3的取反值等于-1减去-3，结果为 2。

对一个整数连续两次“否运算”，得到它自身。
~~3 // 3
所有的位运算都只对整数有效。否运算遇到小数时，也会将小数部分舍去，只保留整数部分。所以，对一个小数连续进行两次否运算，能达到取整效果。
~~2.9 // 2

使用否运算取整，是所有取整方法中最快的一种。

对字符串进行否运算，JavaScript引擎会先调用Number函数，将字符串转为数值。

否运算对特殊数值的处理是：超出32位的整数将会被截去超出的位数，NaN和Infinity转为0。

异或运算
在两个二进制位不同时返回1，相同时返回0。

连续对两个数a和b进行三次异或运算，a?=b, b?=a, a?=b，可以互换它们的值。这意味着，使用“异或运算”可以不引入临时变量，互换两个变量的值。

异或运算也可以用来取整。
12.9^0 // 12

左移运算符（<<）
将一个数的二进制形式向前移动，尾部补0。

如果左移0位，就相当于取整，对于正数和负数都有效。

使用左移运算符，将颜色的RGB值转为HEX值。

右移运算符（>>）
将一个数的二进制形式向右移动，头部补上最左位的值，即整数补0，负数补1。

右移运算可以模拟2的整除运算。

带符号位的右移运算符（>>>）
将一个数的二进制形式向右移动，不管正数或负数，头部一律补0。该运算总是得到正值。对于正数，该运算的结果与右移运算符（>>）完全一致，区别主要在于负数。

特殊数值

Infinity

除了 0除以 0得到 NaN，任意数除以 0，得到 Infinity。

Infinity 可以用于布尔运算。
Infinity 是JavaScript中最大的值（NaN 除外），-Infinity 是最小的值（NaN 除外）。

isFinite函数：返回一个布尔值

NaN（Not a Number），其类型是 number，主要原因是字符串解析成数字出错。

NaN 与任何数（包括 NaN）的运算，都是 NaN。

NaN不等于自己，不全等于自己

isNaN只对数值有效，如果传入其他值，会被先转成数值。
也就是说，isNaN为 true 的值，可能不是 NaN，而是一个字符串。

因此，使用isNaN之前，最好判断一下数据类型。

判断 NaN 更可靠的方法是，利用 NaN 是JavaScript之中唯一不等于自身的值。

ES6 中，isNaN() 成为了 Number 的静态方法：Number.isNaN()

自动类型转换

  • 减号-，乘号*，肯定是进行数学运算，所以转化为number类型。
  • 加号+，可能是字符串拼接，也可能是数学运算，所以转化为number或string
  • 一元运算，如+[]，只有一个操作数的，转化为number类型 　　

与数字进行关系运算时，纯数字字符串被转为数字，null转换为0（null不能进行和0的相等判定），true转换转为1， false转换为0

string 和 number比，string会被隐式转换为number

string 和 string 也能够进行关系运算，比较的就是字符编码顺序。

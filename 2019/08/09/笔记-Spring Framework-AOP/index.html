Agent是执行main方法前的一个Interceptor，Agent代码和main方法在同一个JVM运行，被同一个Classloader装载，被同一个SecurityPolicy和Context管理。

实现一个Agent只需要实现premain方法：public static void premain(String agentArgs, Instrumentation inst)。

Agent类必须打成.jar包，里面的META-INF/MAINIFEST.MF必须包含Premain-Class属性。

AbstractBeanFactory#doResolveBeanClass

这种方式的@Aspect注解的切面类不在BeanFactory内，所以不要加@Component之类的添加进容器。

Spring在加载目标类的.class文件到JVM之前，AspectJ会将切面类中的增强逻辑织入到.class文件中，生成新的.class字节码，所以不需要动态代理就能实现AOP。

在类加载期，注册的ClassFileTransformer读取类路径下META-INF/aop.xml文件中定义的切面类和目标类信息，生成新的.class字节码，然后交给JVM加载。
